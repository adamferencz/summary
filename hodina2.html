Shrnutí HTML a CSS konceptů ze stránky o seriálech

📥 Stáhnout originální stránku

Chceš si projít a prostudovat originální stránku o seriálech?

⬇️ Stáhnout hodina2.html

1. Struktura HTML dokumentu

Každá HTML stránka má základní strukturu:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Document</title>
  </head>
  <body>
    ...obsah stránky...
  </body>
</html>
  • <!DOCTYPE html> - Deklarace typu dokumentu
  • <head> - Metadata stránky (charset, title, styly)
  • <body> - Viditelný obsah stránky

2. Nadpisy a odstavce

Pro strukturování obsahu používáme:

  • <h1> - Hlavní nadpis (největší)
  • <h2> - Podnadpis (menší)
  • <p> - Odstavec textu
<h1>Neljepší seriály</h1>
<h2>Breaking bad</h2>
<p>Breaking Bad je americký televizní seriál...</p>

3. Seznamy

Pro vytvoření seznamu používáme <ul> (unordered list) a <li> (list item):

<ul>
  <li>Breaking bad</li>
  <li>Game of Thrones</li>
  <li>Stranger Things</li>
</ul>
Výsledek:
  • Breaking bad
  • Game of Thrones
  • Stranger Things

4. CSS Styly uvnitř HTML

Styly můžeme psát přímo do <style> tagu v sekci <head>:

<style>
  h2 {
    color: rgb(93, 4, 108);
  }
</style>

Toto nastaví všechny nadpisy <h2> na fialovou barvu.

5. Inline styly

Můžeme stylovat přímo v HTML tagu pomocí atributu style:

<body style="background-color: pink">

Toto nastaví růžové pozadí celé stránky.

6. CSS Třídy (Classes)

Třídy nám umožňují stylovat vybrané prvky. Definujeme je pomocí tečky .:

Příklad 1: Třída .white-title

.white-title {
  color: white;
}

Použití:

<h2 class="white-title">Breaking bad</h2>
Výsledek: Breaking bad (bílý text)
💡 Tip: Třídu můžeme použít na jakýkoliv prvek pomocí class="nazev-tridy"

7. Kontextové selektory

Můžeme stylovat prvky pouze uvnitř určité třídy nebo tagu:

.red-section h2 {
  color: red;
}

Toto nastaví <h2> na červenou barvu pouze uvnitř prvku s třídou red-section.

Použití:

<div class="red-section">
  <h2>Breaking bad</h2> <-- bude červený
</div>
<h2>Game of Thrones</h2> <-- nebude červený
Výsledek:
Breaking bad (červený nadpis v sekci)
Game of Thrones (normální fialový nadpis)

8. Oddělovací čára <hr>

Tag <hr> vytvoří vodorovnou čáru pro oddělení obsahu:

<hr>
Výsledek:

9. Kontejnery - <div>

Tag <div> slouží jako kontejner pro seskupení prvků:

<div class="red-section">
  <h2>Breaking bad</h2>
  <p>Popis seriálu...</p>
</div>

Všechny prvky uvnitř <div> můžeme společně stylovat pomocí třídy.

10. RGB barvy

Barvy můžeme zadávat v RGB formátu (Red, Green, Blue):

color: rgb(93, 4, 108);

Hodnoty jsou od 0 do 255 pro každou barvu:

  • rgb(255, 0, 0) - Červená
  • rgb(0, 255, 0) - Zelená
  • rgb(0, 0, 255) - Modrá
  • rgb(93, 4, 108) - Fialová (z našeho příkladu)

📝 Shrnutí naučených konceptů

  • ✅ Základní struktura HTML dokumentu
  • ✅ Nadpisy <h1>, <h2> a odstavce <p>
  • ✅ Seznamy <ul> a <li>
  • ✅ CSS styly v sekci <style>
  • ✅ Inline styly pomocí atributu style
  • ✅ CSS třídy (classes) pomocí class="nazev"
  • ✅ Kontextové selektory (např. .red-section h2)
  • ✅ Kontejnery <div>
  • ✅ RGB barvy
  • ✅ Oddělovací čára <hr>
🎯 Praktický tip: Rozdíl mezi inline stylem a třídou:
  • Inline: <h2 style="color: white"> - styluje pouze tento jeden prvek
  • Třída: <h2 class="white-title"> - můžeme použít na více prvků najednou
