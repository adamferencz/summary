1️⃣ Nadpisy (headings)

V HTML existují nadpisy od <h1> do <h6>. Čím větší číslo, tím menší nadpis.

<h1>Největší nadpis</h1>
<h2>Menší nadpis</h2>
<h3>Ještě menší</h3>

Největší nadpis

Menší nadpis

Ještě menší

2️⃣ Obrázky

Pro vložení obrázku používáme značku <img> s atributy src (zdroj) a alt (popis).

<img src="kocickaaaaa.webp" alt="kocicka není dostupná" height="80">

3️⃣ Odkazy (linky)

Odkazy se dělají pomocí značky <a> a atributu href.

<a href="https://www.ghb.cz">Velká budova</a>
🏫 Velká budova (otevřít v novém okně)

4️⃣ Odstavce a mezery

HTML ignoruje více mezer a zalomení řádků — vše zobrazuje jako jednu mezeru.

<p>Tento text
má hodně
řádků,
ale HTML je spojí dohromady.</p>

Tento text má hodně řádků, ale HTML je spojí dohromady.

Pokud chceš **ruční zalomení řádku**, použij <br>.

Ahoj,
toto je nový řádek.

Ahoj,
toto je nový řádek.

5️⃣ Barvy a styly

Pomocí atributu style můžeme měnit vzhled prvku (např. barvu textu).

<p style="color: red">Tento text je červený</p>

Tento text je červený ❤️

6️⃣ Linie a nápovědy

<hr> vytvoří vodorovnou čáru, title ukáže nápovědu po najetí myší.

<p title="Tajná zpráva!">Najdi nápovědu!</p>
<hr>

🕵️ Najdi nápovědu – najeď myší!

7️⃣ Speciální znaky

V HTML musíme psát speciální znaky pomocí kódů:

  • &lt; = <
  • &gt; = >
  • &amp; = &
  • &nbsp; = mezera

Např. kód: &lt;h1&gt;Ahoj&lt;/h1&gt; zobrazí →

Ahoj

Jak napsat < a > na české klávesnici?
Stiskni pravý Alt (AltGr) + , pro < a pravý Alt (AltGr) + . pro >.
